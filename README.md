# node_docker_sample

This repository is docker environment for [This repository](https://github.com/You-saku/Nuxt_sample).


## How to use
```
git clone https://github.com/You-saku/node_docker_sample.git
cd node_docker_sample
mkdir app
cd app
git clone https://github.com/You-saku/Nuxt_sample.git
docker-compose up --build -d
```
To check working this container, Please access http://localhost:3000
